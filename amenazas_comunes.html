¿Sabías que?

Los ataques cibernéticos más comunes incluyen el phishing, que engaña a los usuarios para que entreguen información confidencial mediante correos o sitios falsos; el ransomware, que bloquea el acceso a archivos hasta que se pague un rescate; el malware, que abarca programas maliciosos capaces de dañar sistemas o robar datos; y los ataques DDoS, que sobrecargan servidores con tráfico masivo para dejarlos fuera de servicio. Estos métodos representan serias amenazas para la seguridad digital y requieren medidas preventivas constantes.

Phishing

- Técnica de engaño que simula comunicaciones legítimas (como correos o sitios web).

- Busca obtener datos sensibles como contraseñas o información bancaria.

- Variantes: spear phishing (dirigido), smishing (SMS) y vishing (llamadas).

Ransomware

- Tipo de malware que cifra los archivos del sistema y exige un rescate para liberarlos.

- Suele propagarse por correos maliciosos o descargas inseguras.

- Ejemplo famoso: WannaCry afectó a miles de sistemas en 2017.

Malware

- Término general para software malicioso: virus, troyanos, spyware, etc.

- Puede robar información, dañar archivos o controlar el sistema.

- Se instala al ejecutar archivos infectados o visitar sitios comprometidos.

Ataques DDoS

- Inundan un servidor con tráfico falso desde múltiples dispositivos.

- Objetivo: saturar el sistema y dejarlo fuera de servicio.

- Afectan principalmente a sitios web de empresas o servicios públicos.

Caso Gmail (2017): Un ataque masivo simuló invitaciones legítimas de Google Docs. Al hacer clic, los usuarios eran redirigidos a una página falsa de inicio de sesión de Google, donde los atacantes robaban sus credenciales.

Caso WannaCry (2017): Este ransomware se propagó globalmente en cuestión de horas, afectando a más de 200,000 sistemas en 150 países. El Servicio Nacional de Salud del Reino Unido fue uno de los más perjudicados, con miles de citas médicas canceladas.

Caso Stuxnet (2010): Un gusano informático diseñado para sabotear instalaciones nucleares en Irán. Infectó sistemas industriales mediante vulnerabilidades de día cero, causando daños físicos a centrifugadoras.

Caso DynDNS (2016): Una botnet llamada Mirai utilizó dispositivos IoT para lanzar un ataque DDoS masivo contra Dyn, un proveedor de DNS. Esto dejó fuera de servicio a sitios como Twitter, Netflix y Reddit durante horas.
